# Projeto Python Serverless na Vercel

Este repositório contém uma função serverless escrita em **Python**, pronta para ser hospedada na **Vercel**.

##  Como usar

### 1. Instalar CLI do Vercel
```
npm i -g vercel
```

### 2. Rodar localmente
```
vercel dev
```

### 3. Deploy
Basta conectar este repositório no Vercel e clicar em **Deploy**.

 └── index.py
